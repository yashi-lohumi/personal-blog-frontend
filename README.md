# Personal Blog Platform

A modern, editorial-style personal blog platform built using **React**, **Vite**, and **Tailwind CSS**.

This project was developed as part of a startup internship assignment involving **UI design, prompt engineering, and frontend development**.

The platform represents a founder's personal brand website where an individual can publish articles, showcase expertise, and build authority in domains such as technology, AI, startups, leadership, business strategy, and innovation.

---

## Overview

The website combines the experience of a personal portfolio and an editorial publication.

The design philosophy is inspired by:

* Apple
* Medium
* Substack
* Linear
* Notion
* Stripe Editorial

The interface focuses on:

* Clean layouts
* Excellent readability
* Minimalist aesthetics
* Reusable components
* Responsive design
* Professional storytelling

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd personal-blog-frontend
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
