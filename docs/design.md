# Personal Blog Platform Design System

## Project Overview

Design a premium personal blog platform for an individual founder, consultant, and technology thought leader.

The website is a hybrid of:

- Personal Portfolio
- Editorial Publication
- Professional Blog
- Thought Leadership Platform

The website belongs to one individual.

Primary goal:

Build trust, authority, and a strong personal brand while showcasing articles and expertise.

This is NOT:

- A company website
- A startup landing page
- A SaaS dashboard
- A news portal

The experience should feel calm, intelligent, elegant, and timeless.

---

# Design Principles

Prioritize:

1. Personal brand first
2. Editorial readability second
3. Conversion third

Follow this visual hierarchy:

Hero

↓

Authority

↓

Content

↓

Personal Story

↓

Engagement

↓

Conversion

↓

Footer

Every page should feel premium and intentional.

Use generous white space.

Avoid visual clutter.

---

# Style Inspiration

Blend the design language of:

- Apple
- Medium
- Substack
- Linear
- Notion
- Stripe Editorial

Characteristics:

- Large typography
- Clean layouts
- Editorial feel
- Minimal UI chrome
- Strong spacing
- Soft borders
- Subtle shadows

# Animations

Use subtle motion only.

Include:

- Fade in on scroll
- Hover lift
- Underline animation
- Timeline animation
- Smooth page transitions
- Button micro-interactions

Animation duration:

200ms - 300ms

Avoid:

- Bounce animations
- Large parallax effects
- Excessive motion

---

# Icons

Use Lucide React.

Icon size:

20px - 24px

Use icons sparingly.

---

# Images

Use professional editorial photography.

Image styles:

- Rounded corners
- High quality
- Natural lighting

Avoid stock-looking images.

# Accessibility

Maintain WCAG contrast.

Minimum button height:

44px

Add focus states.

Use semantic HTML.

Use aria-labels where appropriate.

---

# Design Constraints

DO NOT USE:

- Dark mode
- Glassmorphism
- Heavy gradients
- Neon colors
- Dashboard UI
- Overuse of icons
- Complex illustrations
- Excessive animations

The final result should feel like a premium founder's personal brand website.
